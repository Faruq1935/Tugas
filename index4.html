Cara Menginstall Windows 7

#CARA MENG-INSTALL WINDOWS 7 DI VIRTUALBOX

1. Pastikan VirtualBox sudah terpasang pada sistem operasi yang kamu gunakan. Jika belum, kamu bisa menginstall di VirtualBox/Download, Juga jangan lupa siapkan file ISO Windows 7

2. Buatlah mesin virtual baru dengan mengklik tombol "New/logo bintang biru". Kemudian isikan nama dengan sesuka kalian di sini saya akan memakai nama "Windows 7". Untuk tipe OS pilih "Microsoft Windows". Untuk version OS pilih "Windows 7 (32-bit)", Untuk jenis arsitektur pilih sesuai file ISO Winodows 7 yang kamu gunakan "disini saya pilih 32-bit". Jika sudah klik "next".

3. Atur alokasi RAM yang akan digunakan oleh Windows 7 di VirtualBox. Jika kamu memiliki RAM komputer yang cukup besar, kamu bisa mengalokasikan RAM lebih besar untuk VirtualBox. Di sini saya akan cuma mengalokasikan RAM sebesar "1024 MB". Jika sudah klik "next".

4. Selanjutnya, pilih "Create a virtual hard disk now". Lalu klik "create".

5. Selanjutnya pilih tipe hard disk, yaitu "VDI (VirtualBox Disk Image)". klik "next".

6. Pilih "Dynamically allocated". Klik "next".

7. Selanjutnya File location and size disini saya tidak akan merubah apa pun, tapi jika kamu ingin mengubahnya silahkan misal dengan menambah atau mengurangi besarnya virtual hard disk, Jika sudah klik "create".

8. Selanjutnya, guest os yang tadi kamu buat akan muncul di deretan guest os. Klik logo "gear". Setelah itu klik "Network". Pilih "Attached to: Internal Network".

9. Masih di seetings kalian klik "Storage". lalu klik "gambar kaset bertuliskan empty." klik 2 lalu pilih file ISO Windows 7 kalian taruh di directory mana. "Jika sudah klik OK".

10. Selanjutnya tinggal jalankan Windows 7 dengan mengklik "Start".

11. Biarkan tampilan seperti ini lalu klik "next".

12. Klik "Install now".

13. Klik "I accept the license terms", lalu klik tombol "next".

14. Tahap selanjutnya, kamu akan melihat dua pilihan. Yaitu Upgrade dan Custom (advanced). Karena kita sedang melakukan instalasi, bukan upgrade, pilih "Custom (advanced)".

15. Selanjutnya akan muncul virtual hard disk yang tadi kamu buat, tinggal klik "next".

16. Selanjutnya adalah menunggu proses installasi selesai. Membutuhkan waktu sekitar 10 sampai dengan 30 menit.

17. Selanjutnya masukan nama kamu dan nama komputer. Kamu bisa menggunakan nama kamu, atau nama apa saja. Klik "next".

18. Saya srankan unutk dikosongi saja, tapi jika kamu ingin mengisinya silahkan kemudian klik "next".

19. Masukkan product key. Jika belum punya, bisa klik tombol "skip".

20. Selanjutnya, untuk mempermudah proses installasi pilih saja "ask me later".

21. Selanjutnya pilih zona waktu kamu. Atau bisa juga biarkan secara default kemudian klik "next".

22. Selanjutnya pilih "Home network".

23. Tunggu sampai proses terakhir selesai.

24. Selesai, "Ughhh selesai juga" kamu bisa menggunakan Windows 7 di VirtualBox.
